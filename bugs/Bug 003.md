# BUG-003 — Resumo Mensal exibe legendas mas não renderiza o gráfico

## Tipo
Bug visual — renderização de componente

## Descrição

O componente **Resumo Mensal** no Dashboard exibe as legendas das categorias corretamente, mas o gráfico em si não é renderizado. A área do gráfico aparece vazia.

## Passos para reproduzir

1. Acessar o Dashboard
2. Observar o bloco **Resumo Mensal** no lado direito da tela

## Comportamento esperado

Um gráfico (donut/pizza) deve ser exibido representando a distribuição de gastos por categoria, acompanhado das legendas.

## Comportamento encontrado

As legendas das categorias aparecem corretamente, mas a área do gráfico está vazia — sem nenhuma visualização.

## Observação adicional

O comportamento é inconsistente — em alguns momentos o gráfico aparece, em outros não, sem nenhuma ação do usuário entre os dois estados. Isso sugere um problema de estado ou de carregamento assíncrono do componente.

## Impacto

🟡 Médio — Não impede o uso do sistema, mas compromete a experiência do usuário e a principal funcionalidade visual do Dashboard.
