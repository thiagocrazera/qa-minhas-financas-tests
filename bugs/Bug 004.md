# BUG-004 — Navegadores detectam a página como inglês e oferecem tradução

## Tipo
Bug de configuração — atributo de idioma

## Descrição

Tanto o Google Chrome quanto o Microsoft Edge oferecem tradução automática da página para português, mesmo o sistema estando inteiramente em português. Isso indica que o atributo `lang` do HTML está configurado incorretamente (provavelmente como `lang="en"`).

## Passos para reproduzir

1. Acessar `http://localhost:5173` no Chrome ou Edge
2. Observar o popup de tradução automática oferecido pelo navegador

## Comportamento esperado

Nenhuma oferta de tradução deve aparecer, pois o sistema já está em português.

## Comportamento encontrado

O popup "Traduzir a página do inglês?" aparece em ambos os navegadores testados (Chrome e Edge).

## O que seria necessário para corrigir

Alterar o atributo `lang` no `index.html` de:

```html
<html lang="en">
```

Para:

```html
<html lang="pt-BR">
```

## Impacto

🟢 Baixo — Não afeta funcionalidade, mas indica falta de atenção à configuração básica do projeto e pode causar confusão ao usuário final.
